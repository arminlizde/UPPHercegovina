Bootstrap Slider

This page runs Bootstrap with Jssor Slider.

It's extremly easy to use Jssor Slider as a compoment of Bootstrap. Given a slider you worked out, to integrate it with Bootstrap, you can just copy javascript and html code and paste it into your page. This page is a simple demo, please view source of this page or download Bootstrap Carousel Slider Example.

Javascript Code

<script type="text/javascript" src="../js/jssor.slider.mini.js"></script>
<script>
    jQuery(document).ready(function ($) {
        var options = {
            ..
        };
            
        var jssor_slider1 = new $JssorSlider$("slider1_container", options);
        ...
    });
</script>
